# ProjectTemplates
基于 Kotlin 的常用的项目工程模板

## 项目模板列表

### Kotlin Application
```
svn export https://github.com/kklongming/ProjectTemplates.git/trunk/kotlin-app

cd kotlin-app

# 运行命令如下
gradle run

# 调试命令如下, 调试端口 5005
gradle run --debug-jvm

```
